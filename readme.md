### SignalR + RabbitMQ

This is an exercise provinding an implementation of SignalR along RabbitMQ.

#### Concepts and tools
* SignalR
* RabbitMQ
* Minimal APIs
* BackgroundServices
* Identity
* Repository
* Unit of Work
* Dependency Injection

To get things working run the following command `docker-compose up --build`
